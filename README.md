## About the Assignment

This assignment contains a set of basic programs based on Data Structures concepts implemented in C. The main aim of this work is to understand how stacks and queues function and how they can be used to solve simple problems in programming.

Each program focuses on a different concept and helps in improving logical thinking as well as coding skills.

---

## Programs Included

1. String Reversal using Stack
   This program takes a string as input and reverses it using stack operations. It helps in understanding how the stack follows the Last In First Out principle.

File: ass2q1.c

2. Balanced Parentheses Checker
   This program checks whether the given expression has balanced parentheses or not. It uses a stack to match opening and closing brackets.

File: ass2q2.c

3. Next Greater Element
   This program finds the next greater element for each element in a given array. It uses a stack to make the process efficient.

File: ass2q3.c

4. Queue Implementation using Array
   This program simulates a simple queue system where elements are added and removed in order. It includes operations like enqueue, dequeue, and display.

File: ass2q4.c

5. Circular Queue Implementation
   This program implements a circular queue using an array. It overcomes the limitations of a normal queue and uses memory more efficiently.

File: ass2q5.c

## How to Run the Programs

First, compile the program using a C compiler:

gcc filename.c -o output

Then run the program:

./output
